### Ответы на контрольные вопросы

1. **Какие типы данных поддерживает SQLite? Как в SQLite можно хранить логические значения и даты?**  
   SQLite поддерживает: NULL, INTEGER, REAL, TEXT, BLOB, NUMERIC.  
   Логические значения хранятся как INTEGER (0 = false, 1 = true).  
   Даты можно хранить как TEXT (ISO8601), INTEGER (Unix timestamp) или REAL (юлианские дни).

2. **Для чего нужен класс SQLiteOpenHelper? Опишите назначение методов onCreate() и onUpgrade().**  
   `SQLiteOpenHelper` — помощник для удобного создания и обновления базы данных.  
   - `onCreate()` — вызывается при первом создании базы, здесь создаются таблицы.  
   - `onUpgrade()` — вызывается при изменении версии базы, используется для миграции (удаление старой таблицы и создание новой).

3. **В чем разница между методами getWritableDatabase() и getReadableDatabase()?**  
   - `getWritableDatabase()` — открывает базу для чтения и записи.  
   - `getReadableDatabase()` — открывает базу только для чтения (более безопасно и экономно).  
   Ошибка при `getWritableDatabase()` может возникнуть, если база повреждена или нет прав на запись.

4. **Что такое Cursor? Как правильно перемещаться по его элементам и почему важно закрывать его после использования?**  
   `Cursor` — интерфейс, предоставляющий доступ к результатам SQL-запроса.  
   Перемещение: `moveToFirst()`, `moveToNext()`, `moveToLast()`.  
   Важно закрывать (`cursor.close()`), чтобы освободить память и избежать утечек.

5. **Что такое ContentValues и для каких операций он применяется?**  
   `ContentValues` — контейнер для хранения пар «имя столбца — значение».  
   Применяется при операциях INSERT и UPDATE.

6. **В чем отличие методов query() и rawQuery()?**  
   - `query()` — структурированный метод с параметрами (безопаснее).  
   - `rawQuery()` — выполняет произвольный SQL-запрос в виде строки (удобнее для сложных запросов).  
   Пример `rawQuery()` с плейсхолдером: `rawQuery("SELECT * FROM students WHERE grade = ?", arrayOf("10"))`

7. **Как обработать ситуацию, когда таблица уже существует, но её структура была изменена?**  
   Увеличиваем `DATABASE_VERSION`. В методе `onUpgrade()` удаляем старую таблицу (`DROP TABLE IF EXISTS`) и вызываем `onCreate()` для создания новой структуры.
